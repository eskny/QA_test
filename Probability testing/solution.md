# Решение
Если монету подбрасывают n раз, а нужно ровно k орлов, вероятность вычисляется через биномиальный коэффициент:
## 𝑃 = 𝐶(𝑛,𝑘)/2^𝑛, 

## где 𝐶(𝑛,𝑘)=𝑛!/𝑘!(𝑛−𝑘)!
 — число способов выбрать k орлов (успех) из n бросков (общее кол-во попыток).

При трёх бросках общее количество возможных исходов равно:

## 2^3 = 8

Биноминальный коэффициент:

## C(3,2)=3!/2!*1!​=3

Вероятность подбросить ровно 2 орла при трех попытках:
## P = 3 / 8 = 0.375 = 37.5%

# Ответ
Вероятность того, что при трёх подбрасываниях монеты выпадет ровно два орла,
равна **3/8** или **37.5%**.
